## Senior Frontend Engineer

I am a bilingual software engineer based in Bogotá, Colombia, with over 10 years of experience building scalable, high-performance web applications. I specialize in React, TypeScript, Angular, and component-driven architecture, with a strong focus on design systems, accessibility, and maintainable frontend platforms.

### Impact Snapshot

- **10+ years** shipping production web applications.
- **6+ companies** across startup and enterprise environments.
- **3+ migration initiatives** led across AngularJS, Angular, and React ecosystems.

### What I Bring

- Product-driven engineering with measurable outcomes.
- Scalable frontend architecture and modern code quality standards.
- Reusable component systems for faster, more consistent delivery.
- Strong collaboration with product, design, and QA in global agile teams.
- Performance and accessibility as default quality bars.

### Professional Journey

- **Senior Frontend Engineer, Proxet** *(Aug 2023 - Present)*  
  Leading Angular-to-React + TypeScript migrations, building reusable components, and improving performance and maintainability.

- **Senior Angular Developer, Newfire Global Partners** *(May 2023 - Aug 2023)*  
  Delivered and maintained Angular 13 features for healthcare products.

- **Senior Software Engineer, Terminal Inc.** *(Jan 2022 - Mar 2023)*  
  Built Angular features and supported embedded Flutter applications in healthcare workflows.

- **Engineer II / Engineer I, Commure** *(Sep 2020 - Jan 2022)*  
  Worked on microfrontend research, reusable component development, and production reliability for React/Angular platforms.
